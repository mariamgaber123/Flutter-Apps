# ⚡ Flutter Provider State Management App

## 📱 Overview
A simple Flutter application that demonstrates **state management using Provider**.  
It shows how multiple widgets can react to shared state changes efficiently.

---

## 🎯 Features
- ⚡ State management using Provider
- 🔄 Real-time UI updates with ChangeNotifier
- 🎨 Simple and clean UI design
- 👶 Beginner-friendly example

---

## 🧠 What This Project Demonstrates
- Using MultiProvider
- Creating multiple models (model, model2)
- Updating UI using Consumer
- Handling state changes with notifyListeners()

## 🚀 Getting Started

### Prerequisites
- Install Flutter SDK

```bash

cd provider_app
flutter pub get
flutter run 
```
